# Купи Яблуко — друк фото на чохлах

Статичний мобільний сайт для переходу з QR-коду на автоматі.

## Як запустити

Відкрийте `index.html` у браузері або завантажте всю папку на GitHub Pages, Netlify, Cloudflare Pages чи звичайний хостинг.

## Де змінити Instagram і Telegram

Відкрийте `config.js`:

```js
instagramUrl: "https://instagram.com/ВАШ_АКАУНТ",
telegramUrl: "https://t.me/ВАШ_АКАУНТ",
```

## Як додати логотип

1. Покладіть файл у `assets/logo/`, наприклад `logo.png`.
2. У `config.js` вкажіть:

```js
logoUrl: "assets/logo/logo.png",
```

Якщо логотип не вказаний, сайт показує зелене яблуко як акуратну заглушку.

## Як змінити фон

1. Покладіть фото у `assets/images/`.
2. У `config.js` вкажіть:

```js
backgroundImageUrl: "assets/images/background.jpg",
```

Щоб залишити світлий фірмовий фон, залиште порожній рядок.

## Як змінити моделі

Відкрийте `models.js`. Моделі згруповані за брендами:

```js
"Apple": [
  "iPhone 15",
  "iPhone 15 Pro"
]
```

Лічильники моделей оновлюються автоматично.

## Open Graph

Замініть `assets/images/og-cover.jpg` власним зображенням 1200×630 px. Так посилання матиме гарний вигляд у месенджерах.

## Файли

- `index.html` — структура сторінки
- `style.css` — дизайн
- `script.js` — логіка списків і налаштувань
- `config.js` — посилання, логотип, фон і тексти
- `models.js` — список моделей

## Покращення у версії V4

- плавна поява елементів без важких ефектів;
